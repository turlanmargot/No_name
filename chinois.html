Le bras de fer chinois est un jeu dont le but est, pour chacun des deux joueurs, de coincer le pouce de son adversaire avec le sien. Il faut coincer le pouce de l'adversaire entre 3 et 5 secondes pour être déclaré vainqueur. Ce jeu se pratique main dans la main, les participants se tenant par les quatre autres doigts repliés en crochet.
